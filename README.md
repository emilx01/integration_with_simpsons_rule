# calculus_integrating_with_simpson-s_rule
Integration using Simpson's rule

Simpson's Rule is a numerical integration method used to approximate the definite integral of a function by fitting quadratic curves, or parabolas, over pairs of subintervals. Unlike basic geometric approximations like Riemann sums or the Trapezoidal Rule, which use flat or linear segments, parabolas adapt to the natural curvature of a function, yielding a significantly higher degree of accuracy with fewer steps. The rule requires an even number of subintervals ($n$), where each pair of intervals defines a parabolic arc over three sequential grid points. Structurally, the formula represents a weighted average of these points scaled by a factor of $\Delta x / 3$. The absolute endpoints of the interval are weighted by one since they sit on the outer boundaries, while the internal points alternate coefficients based on their geometric role: odd-indexed points act as the midpoints of the parabolic arcs and are weighted by four, whereas even-indexed points serve as shared boundaries between adjacent parabolas and are weighted by two. In this piece of code we find definite integral of e^x using Simpson's rule.
